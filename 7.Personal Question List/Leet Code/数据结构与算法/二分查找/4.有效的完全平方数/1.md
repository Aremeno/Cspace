# 4. 有效的完全平方数

## 题目描述

给你一个 正 整数 num 。如果 num 是一个 完全平方数 ，则返回 true ，否则返回 false 。
**完全平方数** 是一个整数，其值等于另一个整数的平方。
**注意**：不允许使用任何内置的库函数，如 sqrt 。

### 示例 1

```
输入：num = 16
输出：true
```

### 示例 2

```
输入：num = 14
输出：false
```

### 提示

- 1 <= num <= 2^31 - 1

力扣题目链接：https://leetcode.cn/problems/valid-perfect-square/

## 题解

```cpp
class Solution {
public:
    bool isPerfectSquare(int num) {
        int l = 1 , r = num ;
        while(l <= r) {
            long mid = l+(r-l)/2;

            if (mid * mid < num) {
                l = mid + 1 ;
            }
            else if (mid * mid > num) {
                r = mid - 1;
            }
            else {
                return true;
            }
        }
        return false;
    }
};
```

很明显，可以用二分查找来解决此题。
