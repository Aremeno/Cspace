# 数据结构与算法

本目录下记录了个人在学习数据结构与算法过程中所做的笔记，内容涵盖但不限于以下内容：

**仍在持续补充中**

- 复杂度分析
- 基本数据结构（如数组、链表、栈、队列、哈希表、数、堆、图等）
- 常用算法（如排序、搜索、分治、回溯、动态规划、贪心等）

本内容初步基于《**Hello算法**》这本书进行的学习和总结。

本书链接：[Hello算法 - 电子书](https://www.hello-algo.com/chapter_hello_algo/)

**注意**：本笔记仅用于个人学习时提高效率所记，以及用于日后复习时能够符合个人思路的参考内容。

**日后若有时间，对C++学习差不多了，会考虑加入《数据结构、算法与应用C++描述》这本书的内容进行补充。**
