# list

list 是 STL 中的双向链表容器。

对于list的底层实现，可以参考数据结构学习笔记中双向链表的实现（C/C++实现双向链表）。

**但是**，STL中的list并不是简单的双向链表实现，它拥有几个非常重要的特性：

- **永远不为空**： list对象创建时会创建一个哨兵节点（sentinel node），这个节点不存储任何数据，只是作为链表的头尾连接点。这样可以简化插入和删除操作的边界条件处理。
- **循环结构**：尾节点的next指针指向哨兵节点，哨兵节点的prev指针指向尾节点，使得链表形成一个循环结构。
- **迭代器统一**：哨兵节点的存在使得begin()和end()迭代器的实现更加统一，begin()指向第一个实际数据节点，end()指向哨兵节点。
