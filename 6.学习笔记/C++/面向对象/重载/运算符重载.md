# 运算符重载

C++允许为自定义类型重载大部分内置运算符，从而使得这些类型的对象能够像内置类型一样使用运算符进行操作。运算符重载通过定义一个特殊的函数来实现，该函数的名称是关键字`operator`后跟要重载的运算符符号。

例子：

```cpp
#include <iostream>
using namespace std;

class Complex {
private:
    double real;
    double imag;
public:
    Complex(double r = 0, double i = 0) : real(r), imag(i) {}
    
    // 重载加法运算符
    Complex operator+(const Complex& other) {
        return Complex(real + other.real, imag + other.imag);
    }
    
    void display() {
        cout << real << " + " << imag << "i" << endl;
    }
};
int main() {
    Complex c1(1.0, 2.0);
    Complex c2(3.0, 4.0);
    
    Complex c3 = c1 + c2; // 使用重载的加法运算符
    c3.display(); // 输出: 4.0 + 6.0i
    
    return 0;
}
```
