# 函数重载

C++允许在同一作用域中定义多个同名函数，这些函数通过参数的类型、个数或顺序来区分，这种机制称为函数重载（Function Overloading）。  
当调用重载函数时，编译器会根据传递的实参类型和数量来选择最合适的函数版本（称为重载决策）。

**重载规则**：

1. **参数类型**：函数的参数类型必须不同。例如，一个函数接受`int`类型参数，另一个接受`double`类型参数。
2. **参数个数**：函数的参数个数必须不同。例如，一个函数接受两个参数，另一个接受三个参数。
3. **参数顺序**：即使参数类型相同，但顺序不同的函数也可以重载。例如，一个函数接受`(int, double)`参数，另一个接受`(double, int)`参数。
4. **返回类型**：函数的返回类型不能作为重载的依据。即使两个函数的参数列表相同，但返回类型不同，它们也不能构成重载。

**示例**：

```cpp
#include <iostream>
using namespace std;

class printOverload {
    public:
        void display(int i) {
            cout << "整数: " << i << endl;
        }

        void display(double d) {
            cout << "浮点数: " << d << endl;
        }

        void display(char c[]) {
            cout << "字符串: " << c << endl;
        }
};

int main() {
    printOverload po;
    po.display(9);          // 调用 display(int)
    po.display(9.99);      // 调用 display(double)
    po.display("Hello");   // 调用 display(char[])
}
```
