# 构造函数与析构函数

## 构造函数（Constructor）

构造函数是一种特殊的成员函数，用于在创建对象时初始化对象的成员变量。它的名称与类名相同，没有返回类型。

例子：

```cpp
class Person {
private:
    string name;
    int age;


// 1. 默认构造函数
Person() {
    name = "Unknown";
    age = 0;
    cout << "默认构造函数被调用" << endl;
}

// 2. 带参数的构造函数
Person(string n,int a) {
    name = n;
    age = a;
    cout << "带参数的构造函数被调用" << endl;
}

// 3. 初始化列表（推荐使用）

Person(string n, int a) : name(n), age(a) {}
};
```

## 析构函数（Destructor）

析构函数也是一种特殊的成员函数，用于在对象销毁时执行清理操作。它的名称与类名相同，前面加上波浪号（`~`），没有返回类型和参数。

例子：

```cpp
class DataProcessor {
public:
    int* data;
    int size;

public:
    DataProcessor(int s) : size(s) {
        data = new int[size];
        cout << "构造函数：分配内存" << endl;
    }

    ~DataProcessor() {
        delete[] data;
        cout << "析构函数：释放内存" << endl;
    }
};
```
