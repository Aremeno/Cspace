# 友元函数

**友元函数不是类的成员函数。**

声明友元函数时，需要在类定义中该函数原型前加上关键字`friend`。友元函数可以访问类的私有成员和保护成员。

例子：

```cpp
#include <iostream>

using namespace std;

class Box {
double width;
public:
    friend void printWidth(Box box);
    void setWidth(double wid);
};

void Box::setWidth(double wid) {
    width = wid;
}

void printWidth(Box box) {
    cout << "Width of box : " << box.width << endl;
}

int main() {
    Box box;

    box.setWidth(10.0);
    printWidth(box);

    return 0;
}
```

## 友元函数存在的意义

友元函数应当谨慎使用，只在确实需要的时候才使用。
