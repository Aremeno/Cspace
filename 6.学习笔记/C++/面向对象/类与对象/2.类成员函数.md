# 类的成员函数

类的成员函数是定义在类内部的函数，可以操作类的任意对象，也可以访问对象的所有成员

成员函数可以定义在类的内部，也可以定义在类的外部

定义在内部的成员函数，默认是内联函数：

```cpp
class Box {
    public:
        double length;   // 长度
        double breadth;  // 宽度
        double height;   // 高度

        // 成员函数定义在类的内部
        double getVolume() {
            return length * breadth * height;
        }
};
```

在类的外部使用范围解析运算符`::`来定义成员函数：

```cpp
double Box :: getVolume() {
    return length * breadth * height;
}
```

调用成员函数是在对象上使用点运算符`.`：

```cpp
Box box1;        // 声明 Box 类的对象 box1
box1.getVolume(); // 调用成员函数 getVolume
```
